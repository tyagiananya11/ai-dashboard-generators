# Conversational AI Dashboard Generator

This project is a prototype that allows non-technical users to create dashboards using natural language queries.

Example:
"Show monthly sales revenue by region."

The system processes the text query, retrieves relevant data, and generates charts automatically.

Tech Stack:
Python
Streamlit
Plotly
Google Gemini API
Note: This is a prototype built for a hackathon demonstration.
## Demo Query Examples

- Show revenue by region
- Show monthly sales trend
- Show top performing product category

## How to Run

1. Install requirements:
pip install -r requirements.txt

2. Run the app:
streamlit run app.py
